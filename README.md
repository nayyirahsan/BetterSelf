# Best Self

A minimalist habit tracker designed to help you stay consistent and become your best self.

## Features

* Track 10 daily habits with a simple tap
* Visual progress ring showing daily completion
* Streak tracking for each habit
* Weekly date strip with progress indicators
* Stats dashboard (active days, perfect days, total check-ins, best streak)
* Smooth animations and mobile-first design

## Habits Included

* Steps
* Water
* Protein
* Gym
* Mobility
* Sleep
* Social Media
* LeetCode
* Projects
* Fruits & Veggies

## How It Works

* Tap a habit to mark it complete
* Progress updates instantly
* Data is saved locally in your browser (no account needed)

## Tech Stack

* HTML
* CSS
* Vanilla JavaScript
* LocalStorage for persistence

## Goal

Build consistency through simple daily actions and visual feedback.
